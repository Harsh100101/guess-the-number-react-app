# Guess Number (React Native)

Simple number guessing game built with React Native and Expo.

- Start screen to pick a number
- Phone (opponent) guesses your number
- Logs previous guesses

How to run locally:

1. Install dependencies

```bash
npm install
```

2. Start Expo

```bash
npm run start
```

3. Run on device or web

```bash
npm run android
npm run ios
npm run web
```

Notes:
- This project was scaffolded with Expo.
- Consider adding CI and linting for a production repo.
